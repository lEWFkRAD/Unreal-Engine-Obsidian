# How to Add FAB Assets to Your Unreal Engine Project

## Overview
Tutorial on using the **Fab Marketplace** (integrated in UE Editor) to download and import free 3D assets for game levels.

## Step-by-Step Guide

### 1. Accessing the Fab Marketplace
- Open **Content Drawer** > Navigate to **Fab** section.
- Log in to your Epic Games account within Fab.

### 2. Finding Free Assets
- Browse categories (e.g., 3D > Environments).
- Apply **Price > Free** filter.
- Recommended: **Paragon Agora and Moonleaf Environment** pack by Epic Games.

### 3. Adding Assets to Your Project
- Click **Add to Project** on asset page.
- Select a version compatible with your UE version.

### 4. Using the Assets
- Imported assets appear in Content folder (e.g., "KiteDemo" subfolder).
- Explore folder structure (Environments > Cliffs, Foliage, Rocks).

## Key Insights
- **Learning Philosophy**: Use free assets for learning rather than creating from scratch.
- **Asset Library Management**: Epic offers free assets every two weeks.
- **Project Organization**: Create your own project folder to avoid mixing assets.
- **Level Design**: Big objects define shapes and guide player path naturally.
