# How to Add Foliage to Your Level in Unreal Engine

## Overview
Tutorial on adding foliage (plants, rocks, trees) to Unreal Engine levels using the Foliage tool.

## Key Concepts

### Accessing the Foliage Tool
- Go to **Modes** and select **Foliage**.
- Options: Paint foliage, select and move foliage, or manage foliage types.

### Finding Foliage Assets
- Browse asset packs (e.g., Paragon Props) for foliage assets.
- Search for "foliage" in the Content Browser.
- Can also use rocks, trees, and other natural elements.

### Painting Foliage
1. Select foliage types you want to paint.
2. Enable multiple types for mixed painting.
3. Paint by clicking and dragging in the viewport.

### Adjusting Foliage Properties
- **Density**: Controls how many instances spawn (e.g., 0.02 for sparse coverage).
- **Brush Size**: Controls the area affected by each paint stroke.
- **Radius**: Controls spacing between instances.
- **Scale Min/Max**: Controls size variation (e.g., 1-3 for natural variety).

### Using Non-Plant Assets as Foliage
- Rocks and other static meshes can be used as foliage.
- Save assets as foliage type first (click Save for each asset).
- Then they appear in the foliage palette for painting.

### Workflow Tips
- Start with low density and increase as needed.
- Use smaller brush sizes for detailed work near paths and structures.
- Mix different foliage types for natural variety.
- Don't over-paint - some empty space looks natural.

## Key Takeaways
- Foliage tool works with plants, rocks, trees, and other natural elements.
- Adjust density, radius, and scale for natural-looking placement.
- Save non-foliage assets as foliage type before using them.
- Start sparse and add more as needed for a natural look.
