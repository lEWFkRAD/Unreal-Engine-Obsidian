# How to Create a Landscape in Unreal Engine 5

## Core Tutorial: Step-by-Step Process

### 1. Entering Landscape Mode and Creation
- Switch from default selection mode to **Landscape Mode**.
- When creating a new landscape, select a size. The tutorial uses **31 Quads** as a good default.
- Click "Create" to generate the landscape in your level.

### 2. Primary Sculpting Tools and Techniques
- **Sculpt Tool**: Left-click and drag on the landscape to raise terrain and create shapes.
- **Flatten Tool**: Used to create flat areas or platforms. Click on a desired height, and other dragged areas will conform to that level.
- **Smooth Tool**: Used to soften harsh edges and peaks for a more natural look.

### 3. Tool Settings for Control
- All major tools have settings for **Tool Strength** and **Brush Size**.
- **Falloff Setting**: Controls edge harshness.
  - **High Falloff**: Creates harsh, defined edges (useful for cliffs).
  - **Medium/Smooth Falloff**: Creates gradual, natural slopes.

### 4. Useful Editor Features
- **View Bookmarks**: Press `Ctrl + 1` (or `2`, `3`, etc.) to set; press number key to jump.
- **Navigation**: Adjust camera movement speed with mouse scroll wheel while moving.
- **Noise Tool**: Adds random values to terrain for more organic shapes. Decrease brush size and strength for finer detail.
- **Erosion Tool**: Available but can sometimes crash.

### 5. Design Philosophy
- Flatten a central play area, then sculpt surrounding mountains as boundaries.
- The landscape can be edited again at any time - don't worry about perfection initially.

## Key Takeaways
1. Start with **31 Quads** and **Sculpt Tool** for initial shaping.
2. Master **Sculpt, Flatten, and Smooth** tools.
3. Constantly tweak **Brush Size, Tool Strength, and Falloff** for desired effects.
4. Use `Ctrl + [Number]` bookmarks for efficient navigation.
5. The landscape is non-destructive and can be returned to at any time.
