# UE5 Tutorial Knowledge Base

An Obsidian vault containing extracted summaries, transcripts, and step-by-step guides from Unreal Engine 5 YouTube tutorials.

## Structure

| Folder | Content | Files |
|--------|---------|-------|
| `UE5_RPG_Framework/` | Gorka Games RPG series — combat, AI, quests, equipment | 13 |
| `Blueprint_Fundamentals/` | Ask A Dev visual scripting — variables, events, loops, UI | 28 |
| `UE5_Beginner_Tutorials/` | Coqui Games coding — classes, functions, interfaces | 17 |
| `UE5_7_Starter_Course/` | Bad Decisions beginner — modeling, lighting, materials | 21 |
| `UE5_PCG_Tutorial/` | PolyBoost procedural — forests, farms, roads | 34 |
| `Learn_to_Code_Blueprints/` | UNF Games landscape/world building | 22 |
| `Step_by_Step_Guides/` | Best tutorials converted to actionable steps | 14 |
| `Individual_Videos/` | Standalone tutorials | 4 |
| `Articles/` | Non-video content | 2 |

## How to Use

1. Open this folder as an Obsidian vault
2. Start from any `_MOC_` file (Map of Content) to browse a series
3. Click `[[wikilinks]]` to navigate between episodes
4. Use tags (`ue5`, `rpg`, `blueprint`, `pcg`, etc.) in search/graph view
5. Graph view shows series clusters naturally

## Tags

- `ue5` — All files
- `rpg` — RPG Framework series
- `blueprint` — Blueprint/visual scripting
- `pcg` — Procedural content generation
- `beginner` — Beginner-friendly content
- `guide` — Step-by-step instructions
- `moc` — Map of Content index files
