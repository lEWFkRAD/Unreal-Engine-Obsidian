# UE5 Beginner Tutorial Part 1: Installation and Using Templates

**Instructor:** Bad Decisions Studio | **Length:** 10:36 | **Published:** 2024-03-04 | **Views:** 626,413

## Overview
Guides through downloading/installing UE5, navigating the Epic Launcher, exploring templates, and understanding key options like Starter Content and Ray Tracing.

## Installation & Setup Steps

### 1. Downloading the Launcher
- Download Epic Games Launcher from https://store.epicgames.com/en-US/download
- Navigate to the "Unreal Engine" tab on the left sidebar

### 2. Navigating the Launcher
- **News:** Stay updated on engine versions (current is 5.3+)
- **Samples:** Download sample projects from Epic to reverse-engineer and learn
- **Marketplace:** Free and paid assets — check "Free for the Month" section monthly
- **Library:** Engine versions, projects, and Vault (downloaded Marketplace assets)

### 3. Installing the Engine
- In Library, click Install on desired engine version
- **Critical:** Click Options before installing — uncheck unnecessary Target Platforms (Android, iOS, Linux) to save ~21-22 GB

### 4. Creating a New Project
- Choose a Template: For clean start, use **Film/Video/Live Events > Blank**
- **Starter Content:** Uncheck to learn importing later
- **Ray Tracing:** Check if you have RTX graphics card
- Project naming: No spaces allowed (use `_` or `-`)
- Initial load compiles shaders — this is normal

## Pro Tips
- Reverse-engineer Epic's Sample Projects to learn
- Download free monthly Marketplace assets consistently
- Templates only add extra functions — the base engine is the same
