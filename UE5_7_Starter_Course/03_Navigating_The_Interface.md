# UE5 Beginner Tutorial Part 2: Navigating the Interface like a PRO

**Instructor:** Bad Decisions Studio

## Overview
Comprehensive guide to UE5 viewport navigation, object manipulation, interface panels, and level management.

## Core Navigation & Controls

### Viewport Movement (WASD + Mouse)
- **WASD:** Forward/Back/Left/Right (FPS-style)
- **E/Q:** Ascend/Descend
- **Right-click + move:** Look around
- **Left-click drag:** Walk/strafe
- **Middle-click drag:** Orbit/pivot view
- **Mouse Scroll:** Adjust camera speed while moving
- **F key:** Focus on selected object

### Object Manipulation
| Tool | Shortcut | Purpose |
|------|----------|---------|
| Move | W | Translation along X, Y, Z axes |
| Rotate | E | Rotation around axes |
| Scale | R | Uniform/non-uniform scaling |

- **Snapping:** Toggle buttons in top toolbar; right-click to configure increments
- **Duplicate:** `Ctrl+D` or `Alt+Drag`

### Key Interface Panels
- **Viewport:** Main 3D workspace
- **Outliner:** Lists all scene objects (visibility toggle with eye icon or `H`)
- **Details Panel:** Properties of selected object (Transform, settings)
- **Content Drawer:** Asset manager (`Ctrl+Space`) — file browser for all project assets

### Bottom Bar Elements
- **Save System:** Bottom-right save icon (saves ALL project changes)
- **Level Selector:** Bottom-left — switch between scenes
- **Content Drawer:** `Ctrl+Space`
- **Simulate/Play:** Bottom-middle for game testing

## Level Management
- **File → New Level:** Basic (clean slate), Empty, or Open World templates
- **Starter Content:** Can add later via Content Drawer → Add → Feature or Content Pack

## Practical Tips
- Save frequently — engine crashes happen
- Organize folder structure from start (Content/Models, Content/Materials)
- `Ctrl+L` + mouse move for sun positioning
