# Post-Apocalyptic Car Wall in UE 5.6 – Part 1: PCG with Splines

**Channel:** PolyBoost | **Published:** 2025-06-14 | **Length:** 29:17

## Overview
Part 1 of a two-part tutorial series recreating a post-apocalyptic scene inspired by El Eternauta. Focuses 100% on structure, layout, and procedural flexibility — no lighting or VFX. Perfect for cinematic scenes or survival games.

**Part 2:** [Cinematic Post-Apocalyptic Street P2](https://www.youtube.com/watch?v=6M9AUl_MAvY)

## What You'll Learn
- Build a massive wall of destroyed cars procedurally using Spline Blueprints
- Populate background with damaged buildings automatically
- Import assets from Fab Marketplace and organize efficiently
- Set up a modular environment for post-apocalyptic worldbuilding

## Key Steps

### 1. Importing Models
- Source assets from Fab Marketplace and CGTrader.
- Free car models: wrecked car, wrecked police car, post-apocalyptic car.

### 2. Creating Blueprint & Spline
- Create Blueprint with spline component.
- Add spline to scene, configure PCG integration.

### 3. Adding Cars to Spline
- Assign car meshes to PCG Static Mesh Spawner.
- Add second car type for variety.
- Create second layer on spline for stacking.

### 4. Fixing Scale Issues
- Address inconsistent mesh scales across different imported models.

### 5. Adding Details
- Add debris and details around the cars using additional PCG nodes.

### 6. Building Blueprint
- Create Blueprint for buildings to populate the background.

## Key Takeaways
- Spline-based car wall: draw the path, PCG stacks and scatters vehicles.
- Layer multiple spline samplers for vertical variety.
- Modular approach: swap assets without rebuilding the PCG graph.
- Part 1 (structure) → Part 2 (lighting, atmosphere, cinematic quality).
